# Addis Ababa University Computer Science Department Website

A simple static website for the Computer Science Department at Addis Ababa University.

## Description

This project is a basic HTML webpage showcasing the department's information, including programs, research, and upcoming events.

## Features

- Responsive navbar with department logo
- Welcome section
- News and announcements
- Academic programs listing
- Quick links sidebar
- Upcoming events
- Footer with copyright
